# Extension Icon Placeholder

The `icons` folder needs three icon files:

- `icon16.png` (16x16 pixels)
- `icon48.png` (48x48 pixels)
- `icon128.png` (128x128 pixels)

## Creating Icons

You can create these icons using:

1. **Online Tools:**

   - [Favicon.io](https://favicon.io/) - Generate icons from text/emoji
   - [RealFaviconGenerator](https://realfavicongenerator.net/)

2. **Design Software:**

   - Photoshop / GIMP
   - Figma / Canva
   - Any image editor

3. **Icon Sets:**
   - Download free icon packs from [Icons8](https://icons8.com/)
   - Use [Font Awesome](https://fontawesome.com/) icons

## Quick Emoji Icon

The easiest way is to use an emoji:

1. Go to https://favicon.io/emoji-favicons/
2. Choose the 🔐 (lock with key) emoji
3. Download the generated icons
4. Rename and copy to this folder:
   - `favicon-16x16.png` → `icon16.png`
   - `favicon-48x48.png` → `icon48.png` (you may need to resize)
   - `android-chrome-192x192.png` → resize to 128x128 → `icon128.png`

## Temporary Workaround

For development, the extension will still work without icons, but will show a default placeholder in the browser.
