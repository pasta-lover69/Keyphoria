Browser Extension Setup

1

Download the Extension

Clone or download the browser-extension folder from the project repository.

2

Open Chrome Extensions

Go to chrome://extensions and enable Developer Mode (top-right toggle).

3

Load the Extension

Click "Load unpacked" and select the browser-extension folder.

4

Pin & Login

Pin the Keyphoria extension from the puzzle icon, then log in with your account.

5

Auto-fill Passwords

Visit any login page — Keyphoria will offer to auto-fill your saved credentials!

⚠️

Not all websites use the same form structure, so auto-save may not always trigger. You can always add passwords manually through the extension or this web app.
